### 설치 방법

* Clone $ install
```sh
git clone ...........
cd data_architecture
~/data_architecture$ pip3 install -r requirements.txt
```

* Append two line below to ```~/.barshrc``` file.
```
~$ cat >> ~/.baschrc
export DATA_ARCHITECTURE=/home/data_architecture
export PYTHONPATH=$PYTHONPATH:$DATA_aRCHITECTURE
Ctrl+d

source ~/.baschrc
```
